# 力学モデルの収束性改善メモ

高密度グラフ（`m / n^2` が大きい）では，頂点間の斥力が急激に大きくなり，力学モデルが剛性の高い（stiff）系になりやすい．
その結果，数値積分が不安定になって振動が収束しないことがある．

## 修正内容

- 速度更新後の値で位置を更新する半陰的オイラー（symplectic Euler）に変更した．
- これにより，大きな斥力が働くときでもエネルギーが暴走しにくくなり，収束性が改善される．

該当箇所: `src/graph/simulator.rs`

## 追加で有効な調整指針

収束が十分でない場合は，以下のいずれかを小さくする（または大きくする）ことで安定化しやすい．

- `dt` を小さくする（時間刻みを細かくする）
- `h` を小さくする（減衰を強める）
- `c` を小さくする（斥力を弱める）
- `max_v` を小さくする（速度の上限を下げる）

これらは `src/config.rs` の `SimulateConfig` で調整する．
